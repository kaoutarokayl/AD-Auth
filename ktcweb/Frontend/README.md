# KTC Web Frontend

Un projet React moderne avec Vite et Tailwind CSS.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

L'application sera accessible sur `http://localhost:5173`

## Build

```bash
npm run build
```

## Stack

- **React 18** - Bibliothèque UI
- **Vite** - Bundler et dev server
- **Tailwind CSS** - Framework CSS utility-first
- **JavaScript** - Langage de programmation

## Structure du projet

```
src/
├── App.jsx        - Composant principal
├── main.jsx       - Point d'entrée
└── index.css      - Styles globaux
```

## Prochaines étapes

1. Personnalisez `src/App.jsx` avec votre contenu
2. Ajoutez des composants dans `src/components/`
3. Créez des pages dans `src/pages/`
4. Adaptez la configuration Tailwind si nécessaire
